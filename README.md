# jsanter27.github.io

Personal professional website for **Justin Santer** — Staff Forward Deployed AI Engineer & Agent Orchestration Lead at [PolyAI](https://poly.ai).

Live at: [jsanter27.github.io](https://jsanter27.github.io)

## Stack

Pure static HTML/CSS/JS — no frameworks, no build step. Hosted on GitHub Pages.

- `index.html` — single-page layout
- `css/styles.css` — dark mode design system with purple gradient accent
- `js/main.js` — particle canvas, typewriter, scroll animations, count-up stats

## Features

- Animated particle network hero with typewriter effect
- Scroll-triggered reveal animations throughout
- Count-up stat cards
- Pulsing timeline marker for current role
- Skill tag stagger animation
- Responsive with mobile nav drawer
